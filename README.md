# casino-bonus MCP Server

Compare crypto casino bonuses, welcome offers, and promotions across platforms.

## 🚀 Deploy in 60 seconds

### Option 1: One-command deploy
```bash
npm install -g mcphosting-cli
mcphosting login
mcphosting deploy --template casino
```

### Option 2: Fork and deploy
1. [Fork this repo](https://github.com/gorlomi-enzo/mcp-casino-bonus/fork)
2. Run: `mcphosting deploy --github https://github.com/YOUR_USERNAME/mcp-casino-bonus`

### Option 3: Run locally
```bash
git clone https://github.com/gorlomi-enzo/mcp-casino-bonus.git
cd mcp-casino-bonus && npm install && npm start
```

## Tools

- **search_bonuses** — Search bonuses by type (welcome, reload, cashback)
- **compare_casinos** — Compare casino offers side by side
- **get_bonus_details** — Get detailed bonus terms and conditions

## Connect to your AI

```bash
mcphosting connect casino-bonus
```
